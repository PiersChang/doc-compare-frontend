您的文件如何被處理？

您上傳或貼入的文件內容，會在您的裝置上解析成純文字，再透過加密連線（HTTPS）傳送給我們的伺服器進行分析。我們不會在資料庫中儲存您的原始文件內容。

AI 分析使用什麼服務？

分析功能使用 OpenAI API。根據 OpenAI 的 API 使用條款，透過 API 傳送的資料預設不會用於訓練 AI 模型，這與 ChatGPT 網頁版的政策不同。詳情請參閱 OpenAI API 資料使用政策。

我們儲存哪些資料？

  • 您的帳號資訊（Email、加密後的密碼）
  • 使用次數記錄（用於計算免費額度與點數）
  • AI 分析結果摘要（用於歷史記錄功能，如已開啟）
  • 不儲存您的原始文件內容

適合哪些文件？

本工具適合一般商業合約、服務條款、採購文件等的版本比對。若您的文件涉及國家機密、高度敏感個資或受特定法規保護的資料，建議先諮詢您的法務或資安團隊。

💡 如有任何資料安全疑慮，歡迎聯絡我們進一步說明。
